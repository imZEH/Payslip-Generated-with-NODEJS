# Payslip-Generated-with-NODEJS

Simple application to Generate Payslip from Excel format to PDF Format.

# Install All Packages in one command
- npm install

# Requirements
- npm install parse-xl
- npm install pdfkit
- npm instlal express

# Run the Application
- set Name= [value]  /** value= INDIVIDUAL / NAME(ex. John) **/
- node index.js
